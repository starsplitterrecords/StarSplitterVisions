# Brand assets

Place global Star Splitter Visions brand images in this folder.

Required filenames:
- `hero.png`
- `hero-mobile.png`
- `social.png`
- `logo.png`
- `icon.png`
- `texture-wide.png`
- `texture-square.png`

Runtime path examples:
- `/images/brand/hero.png`
- `/images/brand/logo.png`

Use `/images/...` in runtime code, **not** `/public/images/...`.
