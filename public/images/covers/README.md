# Cover assets

Place issue/book cover images in this folder.

Naming convention:
- lowercase kebab-case
- include series slug + issue number
- format: `<series-slug>-issue-##.png`

Examples:
- `vikings-2026-issue-01.png`
- `glass-hour-issue-01.png`
- `rex-of-the-splitters-issue-01.png`
- `sunforge-outlaw-issue-01.png`
- `signal-atlas-issue-01.png`

Runtime path example:
- `/images/covers/glass-hour-issue-01.png`

Use `/images/...` in runtime code, **not** `/public/images/...`.
