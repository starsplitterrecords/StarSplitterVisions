# vikings-2026 page assets

Place issue page images for `vikings-2026` inside issue folders.

Folder and naming convention:
- `issue-01/`, `issue-02/`, etc.
- pages named `page-001.jpg`, `page-002.jpg`, ...

Example:
- `public/images/pages/vikings-2026/issue-01/page-001.jpg`

Runtime path example:
- `/images/pages/vikings-2026/issue-01/page-001.jpg`

Use `/images/...` in runtime code, **not** `/public/images/...`.
